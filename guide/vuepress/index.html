# 시작하기

본 페이지는 VuePress를 사용하여 웹 페이지를 구성하는 방법에 대해 가이드 하고 있습니다.
VuePress를 처음 접하신다면 VuePress Introduction페이지를 참고 해주세요.

# 설치방법

VuePress 프로젝트를 설치할 폴더를 생성합니다.

$ mkdir vuepress-test
$ cd vuepress-test
1
2

yarn(or npm) 명령어로 VuePress 모듈을 설치합니다.
전역 설치를 원치 않을 경우 주석을 참고하여 Dev용으로 설치하세요.

$ yarn global add vuepress # OR yarn add vuepress -D
1

WARNING

글로벌 옵션으로 설치했을 경우 프로젝트 내에 아무 파일도 존재 하지 않을 것입니다.
이 경우 기본 프로젝트 구성을 위해 yarn init 명령어를 입력해주세요.

개발 환경 설정을 위해 프로젝트 내부 package.json 파일에 다음 구문을 추가합니다.

{
  "name": "blog",
  "version": "1.0.0",
  ## scripts 부분을 추가합니다
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  },
  "repository": "https://github.com/cookpi/blog.git",
  "author": "cookpi <cookpi@cookapps.com>",
  "license": "MIT",
  "devDependencies": {
    "vuepress": "^0.14.9"
  }
}

# 페이지 구성

# Root Page 구성

우리가 원하는 화면을 보기 위해 페이지 및 라우팅 구성을 합니다.
하단 트리 구조와 같이 docs 폴더를 만든 후 README.md 및 .vuepress 폴더를 생성합니다.
README.md 파일은 웹 서버에서 사용하는 index.html 역할을 합니다.
이 파일은 root로 지정되어 있는 docs폴더의 index 페이지로 사용될 것입니다.

├─ docs
│  ├─ README.md
│  └─ .vuepress
│     └─ config.js
└─ package.json

README.md 및 config.js를 수정합니다. 더 자세한 설정은 VuePress basic-config 섹션을 참고하세요

# README.md
---
home: true
---

# config.js
module.exports = {
  title: 'VuePress',
  description: 'Hello VuePress'
}
